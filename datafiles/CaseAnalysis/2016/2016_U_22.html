Bussa Overseas and Properties Private Limited and another v Union of India and another

Supreme Court of India

22 January 2016

Case Analysis

Bench

Dipak Misra, Shiva Kirti Singh

Where Reported

2016 Indlaw SC 310; (2016) 4 SCC 696; AIR 2016 SC 938; 2016 (332) E.L.T. 197; JT 2016 (2) SC 262; 2016 (6) MahLJ 694; 2016 (4) MPLJ 509; 2016 (1) RCR(Civil) 955; 2016(1) SCALE 495; [2016] 1 S.C.R. 266

Case Digest

Subject: Customs

Keywords: Jurisdiction, Customs Act, 1962, Provisional, Review, India, Representation Of The People Act, 1951, Code of Civil Procedure, 1908, Constitution of India, 1950

Summary: Civil Procedure - Code of Civil Procedure, 1908 - Application for review dismissed - Failure to assail main order - Justification - Appellant filed application for refund of excess provisional customs duty - HC held that as appellant has recovered amount from customers, hence, appellant is not entitled to claim refund - Review application was filed by appellant - HC vide impugned order declined to condone delay in preferring application for review and held that application for review is devoid of merit - Respondent contended that order passed in writ petition having not been assailed, challenge only to order passed in review is not tenable, hence, appeal is not maintainable - Hence, instant appeal.

Held, when prayer for review is dismissed, there is be no merger. If order passed in review recalls main order and different order is passed, then main order does not exist. In that event, there is no need to challenge main order, for it is order in review that affects aggrieved party. SC held that if basic judgment is not assailed and challenge is only to order passed in review, SC is obliged not to entertain such special leave petition. Hence, SC held that appeal is not maintainable. Appeal dismissed.

Ratio - When order passed in writ petition having not been assailed, challenge only to order passed in review is not tenable.
